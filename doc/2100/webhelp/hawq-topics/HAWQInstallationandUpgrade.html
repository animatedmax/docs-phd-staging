HAWQ Installation and Upgrade

This guide provides information and instructions for superusers responsible for installing and configuring a HAWQ system.

This guide assumes knowledge of Linux/UNIX system administration, database management systems, database administration, and Structured Query Language (SQL). Because HAWQ is based on PostgreSQL 8.2.15, this guide assumes some familiarity with PostgreSQL.
