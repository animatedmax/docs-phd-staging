HAWQ - Internal Tables

Overview

In this exercise, you create HAWQ tables and load data into the tables using the COPY command.

Prerequisites

  1. Install the Pivotal HD Single-Node Virtual Machine and start the Pivotal HD services. See Pivotal HD Single-Node VM.

  2. Load the Retail Demo Dataset before following the steps in this section. See Retail Demo Dataset.

Create Internal HAWQ tables

To create HAWQ tables, you use the psql utility to execute SQL commands that create the database and tables in HAWQ.

  1. Start the utility by entering the psql command at the command prompt.

    For example:

    [gpadmin@pivhdsne sample-data]$ psql
    Timing is on.
    psql (8.2.15)
    Type "help" for help.

gpadmin=#

  2. Create the retail_demo schema

    CREATE SCHEMA retail_demo;

  3. To create the tables, you can run the following script that contains all of the SQL statements to create the tables:

    pivotal-samples/hawq/hawq_tables/create_hawq_tables.sql

    For example:

    gpadmin=#\i /pivotal-samples/hawq/hawq_tables/create_hawq_tables.sql

    or you can enter each SQL command separately by entering the following SQL commands:

  • To create the table retail_demo.categories_dim_hawq:

    CREATE TABLE retail_demo.categories_dim_hawq
(
    category_id integer NOT NULL,
    category_name character varying(400) NOT NULL
)
WITH (appendonly=true, compresstype=quicklz) DISTRIBUTED RANDOMLY;

  • To create the table retail_demo.customers_dim_hawq:

    CREATE TABLE retail_demo.customers_dim_hawq
(
    customer_id TEXT,
    first_name TEXT,
    last_name TEXT,
    gender TEXT
)
WITH (appendonly=true, compresstype=quicklz) DISTRIBUTED RANDOMLY;

  • To create the table retail_demo.order_lineitems_hawq:

Load the data into HAWQ tables

In these steps, you data into the tables you just created.

  1. In a terminal session, cd to the /retail_demo directory.

  2. To load the data, copy and paste the following commands into the command window:

    zcat customers_dim/customers_dim.tsv.gz | psql -c "COPY retail_demo.customers_dim_hawq FROM STDIN DELIMITER E'\t' NULL E'';"
zcat categories_dim/categories_dim.tsv.gz | psql -c "COPY retail_demo.categories_dim_hawq FROM STDIN DELIMITER E'\t' NULL E'';"
zcat order_lineitems/order_lineitems.tsv.gz | psql -c "COPY retail_demo.order_lineitems_hawq FROM STDIN DELIMITER E'\t' NULL E'';"
zcat orders/orders.tsv.gz | psql -c "COPY retail_demo.orders_hawq FROM STDIN DELIMITER E'\t' NULL E'';"
zcat customer_addresses_dim/customer_addresses_dim.tsv.gz | psql -c "COPY retail_demo.customer_addresses_dim_hawq FROM STDIN DELIMITER E'\t' NULL E'';"
zcat email_addresses_dim/email_addresses_dim.tsv.gz | psql -c "COPY retail_demo.email_addresses_dim_hawq FROM STDIN DELIMITER E'\t' NULL E'';"
zcat products_dim/products_dim.tsv.gz | psql -c "COPY retail_demo.products_dim_hawq FROM STDIN DELIMITER E'\t' NULL E'';"
zcat payment_methods/payment_methods.tsv.gz | psql -c "COPY retail_demo.payment_methods_hawq FROM STDIN DELIMITER E'\t' NULL E'';"
zcat date_dim/date_dim.tsv.gz | psql -c "COPY retail_demo.date_dim_hawq FROM STDIN DELIMITER E'\t' NULL E'';"

Running HAWQ Queries

Use the psql utility to run the following query on the retail_demo.orders_hawq table. The query returns the top ten postal codes by revenue:

select billing_address_postal_code, sum(total_paid_amount::float8) as total, 
sum(total_tax_amount::float8) as tax
from retail_demo.orders_hawq
group by billing_address_postal_code
order by total desc limit 10;
